# Vehicle Management System
A java console application developed using oop concepts we learned during our class.
This project demonestrates the use of inheritance, abstraction, polymorphism, encapsulation, exception handling, ArrayList and user interaction using scanner.

# Features
- add different types of vsehicles
- diplays vehicle information
- their dpecific behaviors
- store multiple vehicle using arraylist
- exception handling with try-catch
- user input handling using scanner

# OOP Concepts Used
- Encapsulation
- Constractors
- Inheritance
- Method overriding
- Polymorphism
- Abstraction
- Exception handling

# Classes include
- Vehicle(abstract class)
- Car
- TRuck
- Bike
- Main

# How to run
1. open the project in intelljIDEA or any other editor you use
2. compile and run the main class
3. follow the menu instructions in the console.
